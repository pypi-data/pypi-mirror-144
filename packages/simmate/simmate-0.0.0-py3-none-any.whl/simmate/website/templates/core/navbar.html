Welcome to Simmate!


The Simulated Materials Ecosystem (Simmate) is a toolbox and framework for materials science research. It lets you explore various crystal databases, predict new materials, and quickly calculate properties -- electronic, elastic, thermodynamic, and more.

All our software is free and open-source, so making an account is 100% optional. If you'd like to download Simmate's framework and access some advanced tools, take a look at our github page.
